# GEMPAR

Empowering youths for a new era of disruptions to the former norm

## Requirements

- Node.js >= 20

## Setup Guide

Clone the repository

```bash
git clone https://github.com/Hkaar/gematik-2025.git
```

Go to the project directory

```bash
cd gematik-2025
```

Install dependecies

```bash
npm i
```

Run the dev server

```bash
npm run dev
```
